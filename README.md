# MoonCalendar

Interactive web-based calendar with month and year views.  
For each day it will show events, public holidays and basic astronomical data.  
Cells are clickable to display detailed information for a given date.

## Features (planned / in progress)

- Month and year view for browsing dates
- Display of daily events and public holidays
- Display of astronomical data (e.g. moon-related information)
- Clickable cells to show detailed information in a popup or detail view

## Tech Stack

- Backend: Java 17, Spring Boot, REST APIs
- Database: MySQL
- Frontend: React, HTML/CSS/JavaScript
- Tooling: Git, Maven, IntelliJ IDEA


## Python virtual environment (macOS)

- `python3 -m venv .venv`
- `source .venv/bin/activate`
- `python -m pip install -U pip`
- `pip install -r requirements.txt`
